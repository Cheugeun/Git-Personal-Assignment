# 깃허브 마일스톤의 기능
## 마일스톤이란?

• 유사한 이슈들을 하나로 모아 일정관리를 하는 기능

• '스프린트(Sprint)' 개발 방법론을 지원하기 위한 기능

• 엮여 있는 이슈들의 Open, Close를 추적하여 진척상황을 퍼센트로 보여줌
